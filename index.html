Hola, mi nombre es Lucas Heredia, Web Developer

¡Hola! Soy Lucas, tengo 21 años y desde los 13 años, me ha fascinado el mundo digital y su funcionamiento detrás de escena. Siempre me he preguntado cómo se diseña una página web y dónde se almacenan los datos que proporcionamos. Estas preguntas y muchas otras inquietudes despertaron en mí el deseo de adentrarme en el mundo de la programación. Tengo ganas de absorber conocimiento con respecto a la programación y diferentes lenguajes de programación, soy alguien que constantemente busca aprender. Mi objetivo es integrarme a la programación ya que considero que es una tecnologia que ayuda muchisimo a la vida diaria hoy en día.

Mis Proyectos

  • Portafolio

    Portafolio es un sitio web en el cual se puede ver mi portafolio, es mi primer proyecto y al final de este mismo se pueden agregar actividades, pero si se recarga la pagina el contenido se reinicia.

    Nota: Este proyecto fue realizado con: HTML, CSS y JavaScript

  • Cinematic

    Cinematic es un cinema en el cual se le pueden agregar peliculas a una base de datos creada por mi.

    Nota: Este proyecto fue realizado con: HTML, CSS y JavaScript para el Front-end. Express y Mongoose para el Back-end

  • En Lo De Pepe

    En Lo De Pepe es un gestor de turnos donde se puede agendar un turno pero el usuario debe iniciar sesion, para acceder.

    Nota: Este proyecto fue realizado con: React/Vite y JavaScript para el Front-end. Express, TypeORM y TypeScript para el Back-end

  • BuyMe

    BuyMe es un E-Commerce en el cual se puede comprar productos pero el usuario debe poseer una cuenta para acceder.

    Nota: Este proyecto fue realizado con: NextJS, JS-Cookie, TypeScript y Tailwind CSS para el Front-end. Express, TypeORM, JWT y TypeScript para el Back-end

  • Questie

    Repo Front-end

    Repo Back-end

    Questie es una plataforma de aprendizaje en línea diseñada para ofrecer cursos de alta calidad en una variedad de disciplinas. Desde programación y diseño gráfico hasta marketing digital y desarrollo persona. Tiene incluido un chat de OpenAI en conde se puede consultar, metodos de pago con PayPal y Mercado Pago para la compra de monedas que te permiten desbloquear cursos pagos.

    Nota: Este proyecto fue realizado con: NextJS, JS-Cookie, TypeScript y Tailwind CSS para el Front-end. NestJS, Postgres, TypeORM, JWT y TypeScript para el Back-end

Estos proyectos estan ordenados de forma cronologica, del más antiguo al mas reciente.
